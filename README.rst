============================
        Deribit robot!
============================


**1) Clone repository:**


    git clone https://github.com/backdev96/deribit_robot

====================================================

**2) Go to your download folder**

====================================================


**3) Create .env file:**

    CLIENT_ID=your_client_id
    
    CLIENT_SECRET=your_client_secret

====================================================

**4) Deployment**

    docker-compose up
